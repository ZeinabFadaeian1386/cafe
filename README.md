# Coffee Website

## English Version

This project is a simple, modern, and responsive coffee shop website built using **HTML**, **CSS**, and **JavaScript**.  
The website includes multiple sections: Home, About Us, Products, and Customer reviews. It features a fixed navigation bar, smooth scrolling, and a responsive layout that adapts well to different screen sizes including mobile devices.

### Features:
- Responsive navigation bar with menu toggle on small screens  
- Stylish product showcase with images and prices  
- Customer testimonials with star ratings  
- Footer with social media links and contact info  
- Simple and clean design with modern fonts and colors  

### Technologies Used:
- HTML5  
- CSS3 (with Google Fonts and Box Icons)  
- JavaScript (for menu toggle and header shadow effect)
